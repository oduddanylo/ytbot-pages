ytbot – Automated Video Posting Tool for TikTok

ytbot is an automation tool designed to streamline video publishing to TikTok. It integrates with tools like n8n and Vizard to fetch, process, and post videos directly to your TikTok account, saving you time and effort.

Key Features

  • Automated video retrieval from connected sources (e.g., Vizard)
  • Custom caption generation using AI
  • Direct posting to TikTok via TikTok Content Posting API
  • Option to post immediately or schedule drafts

How it works

  1. Connect ytbot to your TikTok account via secure OAuth login.
  2. Set your video source and preferences.
  3. ytbot automatically processes and posts videos to TikTok.
